Tags de Título

As tags de título vão de <h1> até <h6> e são usadas para estruturar os títulos do conteúdo. Quanto maior o número, menor a importância.

Tag <p> - Parágrafo

Serve para escrever textos ou blocos de texto dentro da página.

Tags de formatação

  • <strong>: Deixa o texto em negrito.
  • <i>: Deixa o texto em itálico.
  • <u>: Sublinha o texto.
  • <mark>: Destaca o texto (amarelo por padrão).
  • <small>: Deixa o texto pequeno.
  • <del>: Riscado (texto deletado).
  • <sub>: Texto subscrito (abaixo).
  • <sup>: Texto sobrescrito (acima).

Listas

<ul> cria listas não ordenadas (com bolinhas), e <ol> cria listas ordenadas (numeradas). Cada item é criado com a tag <li>.

Links

<a href="URL"> serve para criar um link clicável.

Tag <hr>

Insere uma linha horizontal para separar conteúdos.

Tag <blockquote>

Usada para destacar citações.

Tag <abbr>

Serve para abreviações. Exemplo: HTML.

Tags de mídia

<img>: Usada para inserir imagens.

<audio>: Usada para inserir arquivos de áudio.

<video>: Usada para inserir vídeos na página.

Tags novas que explorei

<font>: Tag antiga para definir fonte (não recomendada no HTML5).
